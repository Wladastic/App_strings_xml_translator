Instructions:

1. get your first finished strings.xml which you want to translate.

2. copy and paste it into any Texteditor.

3. remove everything except the key, example:
	<string name="settings">Settings</string>
	to: settings
	so you have a list of your values.
3.1. Paste it into "compare.txt"

4. do the same with the Strings, example: <string name="settings">Settings</string>
	to: Settings

5. Copy/paste this list into google Translate https://translate.google.com/

6. Copy the translated Text

7. double click run.bat

8. paste the list in the text that opens.

9. close this window.

10.... wait a few seconds...

11. run.bat gives you the translation!!!

HAVE FUN!
